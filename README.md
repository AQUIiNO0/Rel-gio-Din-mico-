# Relógio Digital

Um relógio digital simples e elegante criado com HTML, CSS e JavaScript.

## Descrição

Este projeto exibe um relógio digital com horas, minutos e segundos em tempo real. O design inclui um fundo gradiente escuro com elementos arredondados e sombras para um visual moderno.

## Tecnologias Utilizadas

- **HTML**: Estrutura da página
- **CSS**: Estilização e layout responsivo
- **JavaScript**: Lógica para atualizar o relógio em tempo real

## Estrutura do Projeto

```
Relogio/
├── index.html          # Arquivo principal HTML
├── assets/
│   ├── css/
│   │   └── style.css   # Estilos CSS
│   └── js/
│       └── script.js   # Script JavaScript
└── README.md           # Este arquivo
```

## Como Usar

1. Clone ou baixe o repositório.
2. Abra o arquivo `index.html` em um navegador web.
3. O relógio será exibido automaticamente e se atualizará a cada segundo.

## Funcionalidades

- Exibe horas, minutos e segundos em tempo real
- Design responsivo
- Fundo gradiente dinâmico
- Elementos com bordas arredondadas e sombras

## Personalização

Você pode personalizar o relógio editando os arquivos CSS e JavaScript:

- `style.css`: Alterar cores, tamanhos e estilos
- `script.js`: Modificar a lógica do relógio ou adicionar funcionalidades
